POI Generation Generate a first civilization backbone with settlements, supporting resource sites, and travel waypoints. Best results come after rivers exist. These POIs save directly to the live map and codex, and generated roads can build on them afterward.

Terrain Preview Preview a full terrain draft locally before saving. Region Style gives the generator a broad map archetype. Sliders still let you push the result away from that archetype. Water Coverage controls how much water appears overall. Coastal Edge controls whether one side or corner of the map opens onto a larger sea. Island Amount makes the generator start with more ocean, then carve land back into it as islands and broken coastlines. Biome Continuity controls how strongly nearby hexes smooth into larger regions. World Scale Compression controls how tightly mountains, deserts, wetlands, forests, and coasts can fit together in a playable fantasy map.
Feature Pass Populate compatible terrain features from the shared rules brain. Base terrain and elevation stay as-is.
Overlay Pass Generate starter roads and rivers using the same saved overlay segments as the manual drawing tools.

These passes add overlays without clearing your hand-drawn work. Use Undo if the result is not what you want.
